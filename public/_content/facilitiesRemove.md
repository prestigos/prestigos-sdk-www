<h3 id="facilitiesRemove">facilitiesRemove
  <code>prestigos.facilitiesRemove()</code>
</h3>

Remove a facility.


| Property    | Type          | Description |
| ----------- | --------------|------------ |
| id          | string        | The facility id.

```javascript
prestigos.facilitiesRemove({
  id            : 'bbbccccddda2a2aa33b'
}).then(...);
```
