<h3 id="commentsRemove">commentsRemove
  <code>prestigos.commentsRemove()</code>
</h3>

Remove a comment.


| Property    | Type          | Description |
| ----------- | --------------|------------ |
| id          | string        | The comment id.

```javascript
prestigos.commentsRemove({
  id            : 'bbbccccddda2a2aa33b'
}).then(...);
```
