<h3 id="layersRemove">layersRemove
  <code>prestigos.layersRemove()</code>
</h3>

Remove a layer.


| Property    | Type          | Description |
| ----------- | --------------|------------ |
| id          | string        | The layer id.

```javascript
prestigos.layersRemove({
  id            : 'bbbccccddda2a2aa33b'
}).then(...);
```
