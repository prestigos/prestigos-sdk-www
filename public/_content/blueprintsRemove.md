<h3 id="blueprintsRemove">blueprintsRemove
  <code>prestigos.blueprintsRemove()</code>
</h3>

Remove a blueprint.


| Property    | Type          | Description |
| ----------- | --------------|------------ |
| id          | string        | The blueprint id.

```javascript
prestigos.blueprintsRemove({
  id            : 'bbbccccddda2a2aa33b'
}).then(...);
```
